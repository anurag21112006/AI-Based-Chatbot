# 🤖 AI-Based Customer Service Chatbot

A simple AI-inspired Customer Service Chatbot built using **Python**, **Flask**, **HTML**, **CSS**, and **JavaScript**. The chatbot can answer common customer queries such as greetings, office timings, payment methods, delivery information, refunds, and more.

---

## 📌 Features

- 💬 Interactive chatbot interface
- 👋 Responds to greetings
- 🕒 Provides office working hours
- 💳 Answers payment-related queries
- 🚚 Gives delivery information
- 💰 Explains refund policy
- 🙏 Responds to thanks and goodbye messages
- 🌐 Simple and responsive web interface

---

## 🛠️ Technologies Used

- Python 3
- Flask
- HTML5
- CSS3
- JavaScript
- JSON

## 🚀 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/AI-Based-Chatbot.git
```

### 2. Navigate to the Project Folder

```bash
cd AI-Based-Chatbot
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the Application

```bash
python app.py
```

### 5. Open Your Browser

```
http://127.0.0.1:5000
```
